    Installing from PPA

    1. Add a PPA to your system's Software Sources:

    sudo add-apt-repository ppa:seant9140/srb2 -y

    2. Update your packages:

    sudo apt-get update

    3. Install SRB2:

    sudo apt-get install srb2 srb2-data

    That is all you need to do! You can now launch and play Sonic Robo Blast 2.
